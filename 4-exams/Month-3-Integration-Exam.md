# Month 3 Integration Exam — AI-Era Bridge Engineer

> **Goal:** Assess full-stack integration skills, system reasoning, AI orchestration, and production-readiness.

**Duration:** 4 hours (self-paced)

**Environment:** Full Django + React project, AI tools optional for guidance only.

---

## Section 1: End-to-End Data Flow (20 pts)

1. Select an existing backend API endpoint.
2. Trace full data flow from DB → API → React frontend → UI.
3. Identify all potential failure points.

**Deliverable:** Flow diagram + annotated report of risks

---

## Section 2: Full-Stack Feature Implementation (25 pts)

1. Build a new feature (e.g., user-generated content with summary).
2. Implement backend API with Django/DRF, including serializers, permissions, and validation.
3. Build frontend component to consume API.
4. Handle errors, loading, and edge cases.

**Deliverable:** Working feature + documentation of edge case handling

---

## Section 3: Authentication & Security (15 pts)

1. Implement JWT authentication.
2. Create role-based access to certain endpoints.
3. Ensure frontend handles token expiration and refresh flow.

**Deliverable:** Auth demo + test cases

---

## Section 4: AI-Orchestrated Feature (20 pts)

1. Integrate an AI layer (e.g., text summarization, sentiment analysis).
2. Process frontend input through AI via backend.
3. Validate AI output and handle failures.
4. Document any security or ethical considerations.

**Deliverable:** Working AI feature + audit checklist + documentation

---

## Section 5: Testing & Observability (20 pts)

1. Write unit and integration tests for backend and frontend.
2. Implement logging and monitoring for the new feature.
3. Simulate failure scenarios and demonstrate system resilience.

**Deliverable:** Test suite + logs/screenshots demonstrating observability

---

**Scoring:** 100 points total. See `grading-rubric.md` for detailed scoring of system ownership, integration, AI review, and production-readiness.
