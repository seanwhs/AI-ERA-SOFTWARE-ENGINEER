# Interview Questions — AI-Era Software Engineer

> **Purpose:** Provide a structured set of technical and behavioral questions for hiring junior, bridge, and senior engineers.

---

## 1. Junior Developer Questions

### Technical

* Explain the difference between `for` and `while` loops.
* What is a function? Can you write one in Python/JS?
* How do you use Git to commit and push changes?
* What is a variable scope?
* Demonstrate using an AI tool responsibly to help write a small snippet.

### Behavioral

* Tell me about a time you learned something new quickly.
* How do you handle errors or bugs you don’t understand?
* Can you explain a small project you built and what you learned from it?

---

## 2. Bridge Engineer Questions

### Technical

* Explain the data flow from frontend to backend and back.
* How do you ensure type safety between frontend and backend?
* Describe a scenario where AI-generated code introduced a bug. How would you catch and fix it?
* How would you design an API for scalability and maintainability?
* Write a test case for a Django REST API endpoint.

### Behavioral

* Describe a situation where you had to integrate multiple systems.
* How do you mentor junior developers while ensuring project deadlines?
* Tell me about a failure in a project and how you resolved it.

---

## 3. Senior Developer Questions

### Technical

* Describe an architecture decision that balanced scalability, cost, and complexity.
* How would you implement a secure authentication and authorization flow for a multi-tenant system?
* How do you integrate AI into production systems while mitigating ethical and security risks?
* Explain your approach to CI/CD and testing strategy for a complex system.
* How would you design a monitoring and observability system for a high-traffic application?

### Behavioral

* Describe a time you led a technical team through a challenging project.
* How do you make decisions under uncertainty when systems or requirements are evolving?
* Give an example of how you influenced engineering culture or best practices in your team.

---

> These questions should be adapted according to the role and used in conjunction with the corresponding hiring rubric for scoring and evaluation.
