## Evaluation Categories

| Category                          | Skills/Indicators                          | Score (0-3) |
| --------------------------------- | ------------------------------------------ | ----------- |
| **Coding Fundamentals**           | Loops, conditionals, basic data structures |             |
| **Syntax & Language Familiarity** | Python or JS/TypeScript                    |             |
| **Problem Solving**               | Can solve simple algorithmic challenges    |             |
| **Version Control**               | Git usage, basic branching                 |             |
| **Communication**                 | Can explain code and decisions             |             |
| **AI Interaction Awareness**      | Basic ability to use AI tools responsibly  |             |

---

## Score Interpretation

| Score | Meaning                                                   |
| ----- | --------------------------------------------------------- |
| 0     | Not demonstrated / unsafe / incomplete                    |
| 1     | Partial understanding, requires supervision               |
| 2     | Competent, minimal guidance needed                        |
| 3     | Strong grasp, able to assist peers, understands reasoning |

---

## Notes for Interviewer

* Junior candidates are **not expected** to design full systems.
* Focus on **learning potential, curiosity, and foundational discipline**.
* AI tool usage should demonstrate responsible engagement, not reliance.
* Use small coding exercises, Git tasks, and verbal explanation to score.

---

> This rubric can be aligned with weekly self-audit results for interns or new hires.
